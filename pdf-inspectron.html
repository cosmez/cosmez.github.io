Get it from Microsoft Store

Overview

PDF Inspectron is a professional-grade desktop application designed for developers and power users who need to look inside PDF files. Unlike standard viewers, Inspectron exposes the raw internal object structure, cross-reference tables, and content streams, allowing for precise diagnostics and surgical repairs of corrupted documents.

Built with efficiency in mind, it handles large files gracefully and offers a suite of common manipulation tools.

Key Capabilities

Deep Inspection

Visualize the PDF node tree, examine object dictionaries, and debug stream data directly.

AI-Ready Context

Integrates with Model Context Protocol to assist AI agents in understanding document structure.

Metadata Management

View and edit XMP metadata, document properties, and info dictionaries.

Document Splitting

Extract ranges or explode documents into individual pages with precise control.

Security & Encryption

Analyze encryption settings and remove owner passwords from accessible documents.

Watermarking

Apply text or image watermarks with transparency and positioning controls.
