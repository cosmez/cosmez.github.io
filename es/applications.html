Experiencia en Consultoría Empresarial

Nota: Los siguientes proyectos fueron desarrollados como arquitecto líder o consultor para diversos clientes empresariales.

Plataforma SaaS de Construcción

Diseño de motor de búsqueda full-text y sistema de distribución. Construcción de pipelines de datos sin dependencias para indexación en tiempo real de millones de planos. Incluía mapeo interactivo y notificaciones de licitación.

Imagenología Médica (PACS)

Implementación de interoperabilidad DICOM y HL7. Creación de sistemas seguros de almacenamiento y recuperación de imágenes médicas, asegurando cumplimiento normativo y alta disponibilidad.

Integración de Hardware

Puente entre .NET y hardware industrial. Integración de dispositivos legado (básculas industriales, cajas registradoras) y microcontroladores modernos (ESP32) en flujos de trabajo conectados a la nube.

Redis de Alto Rendimiento

Uso avanzado de Redis más allá del caché. Aprovechamiento de protocolos internos para estructuras de datos personalizadas y uso de RediSearch para consultas complejas en tiempo real.

Ingeniería de Sistemas Linux

Administración de sistemas Debian para entornos de producción. Compilación de herramientas personalizadas, ajuste del kernel y scripts de despliegue automatizado para rendimiento bare-metal.
